# 📖 Project Overview

The **Smart Fuel Station Intelligence System** is an end-to-end Business Intelligence project designed to simulate and analyze operations across a fuel retail network.

The project demonstrates the complete analytics lifecycle—from synthetic data generation and relational database design to ETL, data modeling, KPI creation, and interactive Power BI dashboards.

It replicates how enterprise fuel companies monitor station performance, fuel sales, customer behavior, employee productivity, and business revenue.

---

# 🎯 Objectives

- Generate enterprise-scale synthetic business data
- Design a relational database using SQLite
- Perform ETL using Power Query
- Build a Star Schema data model
- Create reusable DAX measures
- Develop interactive Power BI dashboards
- Generate business insights using KPI-driven analytics
